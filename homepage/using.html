利用ツールの説明

The W3C Markup Validation Service
The W3C Markup Validation Serviceの名前の中のW3Cとはworld wide web Consortiumの略であり、 このW3Cという会社が提供している、XHTML,HTMLコードの文章をチェックしてくれるサービスある。 このコードチェックサービスを利用することにより、ページがインデックスされているのが遅くなったりするなどの 時間や機会のロスなどを防ぐことができる。自分でもコードの確認をしたいとしても、自分の気づかないミスを検出することができるため The W3C Markup Validation Serviceを利用したチェックは重要なのである。
W3C CSS 検証サービス
W3C CSS 検証サービスの名前の中のW3Cとはworld wide web Consortiumの略であり、 このW3Cという会社が提供している、CSSコードの文章をチェックしてくれるサービスある。CSSの仕様と比較するだけでなく、 タイプミスや、CSSが効率悪く書かれている場合、警告、エラーで教えてくれるものである。このサービスを利用することにより、 効率のよく、美しいコードへと導いてくれるのである。
